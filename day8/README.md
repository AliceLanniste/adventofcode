`DAY 8`给出的问题刚开始没弄明白，给出如下的例子,其中`A-B-C-D`组成树结构。
```
2 3 0 3 10 11 12 1 1 0 1 99 2 1 1 2
A----------------------------------
    B----------- C-----------
                     D-----
```


节点由子节点个数和数据两部分组成，以根节点`A`来说，2表示有2个子节点，3表示有3个数据分别是1,1,2。
解决方式就是重点构造如下树结构，暂时不管`metadata`,然后不断递归。
```
        A
      /   \
    B       C
             \
               D
```
